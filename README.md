## Setup

1. Clone the repository

```bash
git clone github.com/nguyendhst/web-automation-testing
```

2. Install dependencies

>**Note:** Run `checks.sh` for a quick check of dependencies and installation

```bash
pip3 install -r requirements.txt
```
